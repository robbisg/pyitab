Utilities package
====================
This package gathers different utility function.

Utilities modules
-----------

* :doc:`atlas`: extract information from standard templates, electrode arrangement and so on. 
* :doc:`bids`: includes functions to manipulated filenames and directories in BIDS format.
* :doc:`dataset`: includes functions to cope with pymvpa dataset format.
* :doc:`files`: general filesystem operations.
* :doc:`image`: basic operations on nibabel images.
* :doc:`math`: math functions.
* :doc:`matrix`: matrix based operations.
* :doc:`time`: simple time utilities.
